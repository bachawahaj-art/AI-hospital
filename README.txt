APS BOYS HOSPITAL – AI POWERED HEALTHCARE SYSTEM
===============================================

PROJECT DESCRIPTION
-------------------
This is a front-end based hospital management and AI consultation system.
It includes:
- AI-based medical diagnosis
- Online doctor appointment booking
- AI lab report analysis
- Medicine home delivery
- Easypaisa & JazzCash payment simulation

TECHNOLOGIES USED
-----------------
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Responsive UI Design

PROJECT STRUCTURE
-----------------
APS-Hospital/
│
├── index.html          → Main website file
├── css/
│   └── style.css       → All styling
├── js/
│   └── script.js       → All JavaScript logic
├── assets/
│   └── images/         → Logos & images
│       ├── Easypaisa-logo.png
│       └── jc.png

HOW TO RUN THE PROJECT
----------------------
1. Open the folder in VS Code
2. Install "Live Server" extension
3. Right-click index.html
4. Click "Open with Live Server"

IMPORTANT NOTES
---------------
- This project is for educational/demo purposes only
- AI diagnosis and payments are simulated
- Not intended for real medical or financial use
